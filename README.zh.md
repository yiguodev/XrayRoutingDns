## 发起 PR

若您有好的分流方案，请按照以下方式提交 PR：

1. 请确保您的 **xray://** 链接可以被成功导入；

2. 添加简明扼要的介绍，格式为 `//介绍`。

如

```
//全局代理，不分流
xray://xxxxxx
```
